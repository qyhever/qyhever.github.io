js中的数组方法

发表于 2017-10-21 | 分类于 js
字数统计: 2,786 字 | 阅读时长 ≈ 12 分钟

前言

Array类型是js中除了Object类型最常用的类型，js中的数组与其他传统语言中的数组有着相当大的区别。

OO语言中的常规数组：起始地址+长度构成的一片连续的地址空间。

js中的数组：其实是对象，只是会自动管理一些”数字”属性和length属性。

在开发过程中，我们还是习惯把它当做多个元素的列表集合，下面总结下数组中的方法。

创建数组

创建数组的基本方式有两种：使用构造函数和字面量语法

使用Array构造函数

在使用Array构造函数时，可以省略new操作符

使用字面量语法

可以看到使用字面量语法比构造函数语法更为简洁，这也是现在创建数组的主流方式。

第3行声明的数组通常称为稀疏数组，不建议这样创建，在IE中认为空位是undefined，在其他浏览器中可能会忽略空位，但长度还是3。

数组方法

检测数组

  • instanceof操作符

    这种方法不能在有多个全局执行环境中使用，会导致多个Array构造函数，从而出现错误。

    ​

  • Array.isArray()

    准确检测数组，不管是在哪个全局执行环境中创建的。

    ​

  • constructor

    每个实例对象都有一个constructor属性，指向创建它的构造函数。

  • Object.prototype.toString.call()

    通用检测方法

    ​

转换方法

所有对象都有toLocaleString()、toString()、valueOf()方法。

  • toString()：返回有数组中每个值的字符串形式拼接而成的一个以逗号分隔的字符串
  • toLocaleString()：通常情况下和toString()一样
  • valueOf()：返回的数组本身

  • join()：使用自定义的分隔符来分隔数组

如果数组中的某一项的值为null或undefined，那么该值在上述转换方法返回的结果中以空白字符串表示。

增删方法

  • push()：后面添加，返回数组长度（可以接受任意数量的参数）
  • unshift()：前面添加，返回数组长度（可以接受任意数量的参数）
  • pop()：后面删除，返回被删除的元素（一次只能删除一个）
  • shift()：前面删除，返回被删除的元素（一次只能删除一个）

排序方法

数组中提供两个可以直接用来进行排序的方法：reverse()和sort()，这两个方法很方便，不过原生for循环也是能够做到的。

  • reverse()：翻转数组

  • sort()：接受一个比较函数来进行排序

    • 对于数值类型

    • 通用升序

    • 通用降序

reverse()和sort()方法返回的都是经过排序之后的数组。

操作方法

  • concat()：连接数组，返回新构建的数组

    • 没有传递参数，复制当前数组并返回副本

      相当于是克隆（浅拷贝），如果元素为引用类型，修改则会影响到原来的数组

    • 多个参数，将这些数组的每一项添加到结果数组中，如果不是数组，也会被添加到结果数组末尾

  • slice()：基于当前数组中的一项或多个项创建一个新数组（截取数组）

    • 没有参数，与concat()不传参类似，浅拷贝

    • 一个参数，以这个参数为起始位置，截取到数组最后

    • 两个参数，表示起始位置和结束位置，[起始，结束)，包头不包尾

  • splice()：最强大的数组方法，删除、插入、替换三种不同的用法。

    • 删除：删除任意数量的项，指定2个参数：要删除的第一项的位置和要删除的项数

    • 插入：插入任意数量的项，指定3个参数：起始位置、0（要删除的项数，0代表不删除）、要插入的项。如果插入多项，可以在第三个参数后面传入

    • 替换：替换任意数量的项，指定3个参数：起始位置、要删除的项数、要替换的项（删除的项数不用与要替换的项数相等）

位置方法

  • indexOf()：查找某一项，两个参数：要查找的项、查找的位置（不传则从第一项开始找），返回要查找的项的位置

    如果查找不到返回-1，即使有多个相等，也只会返回第一个的位置。

  • lastIndexOf()：与indexOf相反，从后往前查找

迭代方法

ES5提供了5个迭代方法，每个方法都接收两个参数：在每一项上运行的函数（可选的）、运行该函数的作用域对象（即在该函数内的this指向）。传入的函数接收3个参数，数组项的值、索引、数组本身。

  • forEach()：对数组中的每一项运行给定函数，没有返回值

  • map()：对数组中的每一项运行给定函数，返回每次函数调用的结果组成的数组

  • filter()：对数组中的每一项运行给定函数，返回该函数会返回true的项组成的数组

  • every()：对数组中的每一项运行给定函数，如果该函数对每一项都返回true，则返回true

  • some()：对数组中的每一项运行给定函数，只要有一项返回true，则返回`true

    第二个参数为函数内部的this对象，不传为window，严格模式为undefined

    ​

    forEach()本质上和普通的for循环没什么区别，只是换了个写法而已

    ​

    filter()可以用来过滤掉数组中的某些项，这也是开发中用的比较多的

    ​

    map()可以用来改变数组中的每一项（映射）

    ​

    every()查询数组中的所有项是否满足条件，有一个为false，则返回false

    ​

    some()查询数组中的某一项是否满足条件，只要有一个为true，则返回true

这些数组方法通过不同的操作，可以更方便的处理数组。

缩小方法

缩小数组的方法：reduce()和reduceRight()，这两个方法都会迭代数组的所有项，返回最终的值。reduce()从数组第一项遍历到最后，reduceRigth()则相反。

这两个方法接收两个参数：在每一项上运行的函数（可选的）、作为缩小基础的初始值。函数接收4个参数，前一项、当前项、索引、数组本身。函数的返回值作为前一项（第一个参数）自动传给下一项。

使用reduce()可以轻松实现数组求和

可以传入第二个参数作为初始值

使用reduce()还是reduceRight()，主要取决于要从哪头开始遍历数组，除此之外，它们完全相同。
