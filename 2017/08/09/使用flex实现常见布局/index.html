使用flex实现常见布局

发表于 2017-08-09 | 分类于 css
字数统计: 1,045 字 | 阅读时长 ≈ 5 分钟

flex布局

flex布局又名为“弹性布局”，通过设置的属性值自动调整元素的宽度和高度值，可以简便、完整、灵活的实现各种页面布局。

在移动端开发中，flex基本已成了主流的布局方式。

flex实现常见布局

圣杯布局

圣杯布局是一种三栏布局，左右两侧定宽，中间自适应的布局。

很简单的就实现了圣杯布局，代码简洁清晰。

经典的上中下布局

上中下布局在网站开发中很常见，这里还要加入sticky-footer布局

sticky-footer：当页面内容高度小于可视区域高度时，footer 吸附在底部；当页面内容高度大于可视区域高度时，footer 排在 content 下方

上中下布局、中间左侧定宽

同样是上中下布局，中间部分左侧定宽，中间部分右侧自适应，常见的后台系统页面布局

左侧定宽、右侧上中下布局

左侧高度自适应，宽度固定，右侧上中下布局，也是常见的后台系统页面布局

sticky-header

header 固定在顶部，不会随着页面滚动。

sticky-sidebar

左侧 sidebar 固定在左侧且与视窗同高，当内容超出视窗高度时，在 sidebar 内部出现滚动条。左右两侧滚动条互相独立。

上面使用flex轻松实现了常见的布局，对于移动端和不需要太多兼容性的后台管理系统，可以放心使用。
